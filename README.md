# Trichology for GP

A complete, evidence-based clinical guide to hair and scalp disorders for general practitioners. Each chapter is a standalone, interactive HTML file covering diagnosis, examination, trichoscopy, and treatment of the most common conditions seen in primary care.

## How to Use

1. Open `index.html` in any browser to access the full table of contents
2. Click any chapter to open it — each file is self-contained
3. Use the built-in Anki-style flashcard decks for spaced repetition
4. All SVG diagrams are scalable and print-friendly

## Technical Details

- Self-contained HTML files with no external dependencies (except Google Fonts on first load)
- Works offline after initial font load
- SVG-based diagrams throughout
- Vanilla JavaScript — no frameworks required
- Mobile-responsive design
- Tested in Chrome, Firefox, Safari, and Edge
